Project 4

Button Functionality:

Start Service : To start the Music Central Service and bind to it.
Stop Service : To unbind from the Music Central Service.
Get All Songs : Fetches all the songs from the service (uses AIDL file) and populates the listView.
Get Specific Song : Fetches specific song info ( the selected radio button ) and opens a pop up window with the song information.
Pause/Resume : Pauses/Resumes the playing song
Stop : Stops the playing song

ListView Functionality:

On click of list view item : Starts playing the clicked song
On selecting the radio button : Click get specific song Button after selecting a radio button to get song info.
